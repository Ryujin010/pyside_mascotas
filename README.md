# 🐾 PySide Mascotas

Sistema de Adopción de Mascotas desarrollado con PySide6, SQLAlchemy y MySQL.

## 📋 Características

- **CRUD completo** de mascotas (Crear, Leer, Actualizar, Eliminar)
- **Interfaz moderna** con tema oscuro (QDarkStyle)
- **Pestañas** para separar listado y estadísticas
- **Búsqueda en tiempo real** por nombre o especie
- **Paginación** (10, 20, 50, 100 items por página)
- **Estadísticas visuales** con tarjetas y gráficos de progreso
- **Exportación** a CSV y PDF con formato profesional
- **Base de datos MySQL** con SQLAlchemy ORM
- **Migraciones automáticas** con Alembic

## 🚀 Tecnologías Utilizadas

- **Python 3.13**
- **PySide6** - Interfaz gráfica Qt
- **QDarkStyle** - Tema oscuro profesional
- **SQLAlchemy** - ORM para base de datos
- **MySQL** - Base de datos relacional
- **Alembic** - Migraciones de base de datos
- **PyMySQL** - Conector MySQL
